# About CircuitTrix Page

## Overview
A dedicated, comprehensive page explaining the CircuitTrix technical event in detail.

## Page URL
`/about-circuittrix`

## Page Sections

### 1. **Hero Section**
- Event introduction
- Brief tagline explaining the core focus on ECE

### 2. **What is CircuitTrix?**
- Detailed description of the event
- Focus on analog and digital circuits
- Emphasis on practical problem-solving

### 3. **Event Objectives**
- Three key goal cards:
  - Test Knowledge
  - Creative Problem-Solving
  - Enhance Skills

### 4. **Event Concept**
- How the event works
- Tools used (Multisim, Proteus, Tinkercad)
- Task structure

### 5. **Event Structure & Timeline**
- **Round 1: Design & Simulation**
  - 30-minute circuit design challenge
  - Projects like temperature alarms, signal amplifiers, logic controllers
  
- **Round 2: Debug the Circuit (Final Showdown)**
  - Fault-finding challenge
  - Identify and fix errors in provided circuits

- **Team Details**
  - 1-2 participants per team

### 6. **Judging Criteria**
Four weighted parameters with visual progress bars:
- **Circuit Accuracy** (40%)
- **Innovation & Creativity** (25%)
- **Documentation & Clarity** (20%)
- **Time Management** (15%)

### 7. **Requirements**
- Computers with simulation software
- Projector for demonstrations
- Timer and quiz sheets

### 8. **Prizes & Recognition**
- **Winner**: Cash prize + Certificate of Excellence
- **Runner-up**: Cash prize + Certificate of Achievement
- **All Participants**: E-Certificate of Participation

### 9. **Call to Action**
- Registration button to encourage sign-ups
